# IntellMeet — Frontend

AI-powered enterprise meeting and collaboration platform built with React + Vite.

## Overview

IntellMeet is an AI-powered enterprise meeting and collaboration platform developed using the MERN stack to enhance communication and productivity for modern remote and hybrid teams. The platform provides real-time video conferencing, live chat, screen sharing, AI-generated meeting summaries, automatic transcription, smart action item extraction, and collaborative team workspaces within a secure and scalable environment.

## Tech Stack

- **React 19** — UI framework
- **Vite** — Build tool with HMR
- **Socket.io Client** — Real-time communication
- **WebRTC (simple-peer)** — Peer-to-peer video/audio
- **Axios** — HTTP client
- **React Router** — Client-side routing

## Getting Started

```bash
npm install
npm run dev
```
